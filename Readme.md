# 🔥 Biometric Attendance System

Production-ready attendance platform integrating **ZKTeco biometric devices** via **ADMS**, with backend deployment on Railway and frontend deployment on Vercel.

---

## 🌐 Live Demo

https://attendance-sigma-one.vercel.app

---

## ⚡ Key Features

* ZKTeco Device Integration
* ADMS Attendance Log Sync
* Employee Attendance Dashboard
* Payroll Support
* Excel Data Handling
* Cloud Deployment
* Frontend + Backend Architecture
* Real-world Production Use
